Do you know who holds the keys in your organisation?

hr

Maintain full control over key management and allocation with the Kyoto Key e-Ledger software. The importance and security significance of this task is often overlooked and many organisations, despite having complex and expensive master key systems, have poor record keeping of who has which keys.


Why do we need the Kyoto Key e-Ledger software?

hr

Kyoto Key e-Ledger (or Key Manager as its sometimes called) provides a comprehensive key management mechanism for all keys within the organisation, those that belong to a master key system and those that are separate such as vehicle keys and locker keys.


Designed as a complimentary product for our own internal locksmithing software, the Kyoto Key e-Ledger can also be used as a standalone product. This is a premium quality program, developed in accordance with industry standards and requirements for this type of software. It features the latest in programming and development technology, which provides ease of use and program reliability and stability.



This software is ideal for:


  • Schools
  • Manufacturing Companies
  • Real Estate Agents
  • Local Councils
  • Corporations
  • Hospitals
  • Sports Clubs
  • Utility companies
  • Aged Care facilities

Its features:

  • Easily manage key issues and returns
  • Control overdue keys
  • Print detailed reports of the who, what, where and when of keys, doors and key-holders
  • Easy to install, configure and use
  • Multi-user
  • Full history reporting
  • Comprehensive step-by-step instructions
  • Unlimited master-key system size
  • Locksmith supported systems
  • Stand-alone systems

Manage multiple systems and sites from one location

hr

Kyoto Key e-Ledger allows for management of multiple systems and even multiple sites from the one location. Better still, it allows Kyoto Security Solutions to 'export' all relevant details from our system into your Kyoto Key e-Ledger , a highly professional and extremely convenient service.


Whether the master key system or systems are imported from us or entered into the program directly, the database records all relevant details including keys, doors, key-to-door assignments (which keys open which doors), signatories, key holders, system address details and more. The highly innovative database links these core pieces of information together in such a way that once set up, the process of managing the keys is extremely simple.
